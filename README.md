## 📖 Descripción

**Book JavaScript Study** es un repositorio creado como parte de mi aprendizaje autodidacta de **JavaScript**, basado en un libro de programación que completé de forma independiente.

El objetivo principal del proyecto es reforzar los conceptos teóricos mediante práctica real, implementando cada ejercicio y ejemplo directamente en código.

Este repositorio refleja mi progreso mientras aprendía:

* Fundamentos de JavaScript
* Variables y tipos de datos
* Funciones y métodos
* Objetos y estructura del lenguaje
* Manipulación básica del entorno web

---

## 🎯 Objetivos del proyecto

* 📚 Aprender JavaScript desde la base mediante un libro técnico
* 💻 Practicar escribiendo código real en lugar de solo leer teoría
* 🧠 Comprender conceptos fundamentales del lenguaje
* 🚀 Desarrollar hábitos de aprendizaje autodidacta

## 🧱 Estructura del proyecto

```
Book-JavaScript-Study/
│
├── 01-myFirstJS/                # Primeros ejercicios y contacto con JS
├── 02-Variables/                # Variables y tipos de datos
├── 03-Functions-Methods-Objects/# Funciones, métodos y objetos
└── javascript-and-jquery-book-code.zip
```

Cada carpeta representa un capítulo o sección del libro y contiene los ejercicios desarrollados durante el estudio.

## ▶️ Cómo ejecutar los ejercicios

1. Clona el repositorio:

```bash
git clone https://github.com/NuriaPines/Book-JavaScript-Study.git
cd Book-JavaScript-Study
```

2. Abre cualquier carpeta del capítulo.

3. Ejecuta los archivos `.html` directamente en el navegador.

No requiere instalación adicional ✅
